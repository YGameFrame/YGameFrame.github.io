详细描述

帧缓冲处理实体

概述：该对象应该被添加到备有帧缓冲的场景YFBOScene （或其子类）中，该对象将整个场景视为一幅纹理YTexture，并对其处理，然后渲染到屏幕上以完成场景级特效。
