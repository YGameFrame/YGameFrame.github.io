详细描述

备有帧缓冲的场景

概述： 该场景备有可供离屏渲染的帧缓冲对象，您可以将此场景视作纹理YTexture 进而对其重新处理，以做出场景级别的特效（例如场景切换）。您应该通过复写帧缓冲处理实体YFBODomain 的子类并将其添入该场景来控制其特效逻辑。

YTexture ygame.framebuffer.YFBOScene.asTexture ( )

将场景视为一幅纹理YTexture，从而您可以对整个场景做其他处理
注：您应该首先调用enableSceneAsTexture(boolean) 来开启该模式，否则将抛出异常。值得注意的是，当场景处于 “正在进入”、“正在退出”状态下，该模式是默认开启，“运行状态”下则是默认关闭的。
