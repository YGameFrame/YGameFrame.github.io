YStateMachineTemplate<S, R, C> ygame.state_machine.builder.YStateMachineBuilder< S, R, C >.buildTransitionTemplate ( )

构建一个状态转换的流程模板，对于众多状态机，如果它们的状态转换流程相同， 那么它们可以共享此模板 ，但它们各自的状态是由自己维护的。当你需要数量庞大却拥有相同状态转化流程的状态机时 ，这是一个高效利用内存的方案。

返回
使用该构建器配置出的状态转换流程模板
