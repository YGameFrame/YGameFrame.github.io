抽象着色程序

概述：使用该类时，您不需要复写onInitialize(int) 方法。此外在对着色程序赋值时（ applyParams(int, ygame.framework.domain.YReadBundle) ），可以利用更为方便的setXXX（如： setUniformf(String, float)等）方法实现。
