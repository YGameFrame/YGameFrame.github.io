void ygame.framework.core.YSystem.switchScene ( YScene  sceneTo,
YIResultCallback  switchResultCallBack 
)

切换场景
注：切换场景时一个异步过程，不会立即返回，如果您需要知道切换的结果，可以通过传入一个回调 YIResultCallback，通过将 YIResultCallback#onResultReceived(Object) 中的参数objResult强制类型转为Boolean对象得到切换结果：真为切换成功，反之失败。
示例：

system.switchScene(mainScene, new YICallback()
{
    public void run(Object objResult)
    {
        if((Boolean)objResult)
            //如果场景切换成功，如此这般……
        else
            //如果失败，如此这般……
    }
});
参数
sceneTo场景
switchResultCallBack切换结果回调
